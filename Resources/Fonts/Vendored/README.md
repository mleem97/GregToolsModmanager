# Vendored Fonts

Place vendored font files here:

- `Inter/` (e.g. `Inter-Regular.ttf`, `Inter-Bold.ttf`)
- `InterTight/` (e.g. `InterTight-Regular.ttf`, `InterTight-SemiBold.ttf`)
- `Nexa/` (e.g. `Nexa-Regular.otf`, `Nexa-Bold.otf`)

Notes:

- Keep only properly licensed font files in this directory.
- Files in these folders are automatically included by the MAUI project configuration.
