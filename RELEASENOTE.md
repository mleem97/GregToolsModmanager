:package: **gregModmanager v1.0.3 is LIVE!**

Hey everyone,

The desktop companion for the Greg Framework has been updated! **gregModmanager v1.0.3** is now available, featuring improved stability and better integration with the latest `gregCore` builds.

---

:wrench: **What's new in v1.0.3?**

:rocket: **Optimized Installation** — Faster deployment of mods to the game directory.
:shield: **Integrity Checks** — The manager now verifies that `gregCore` and its dependencies (Jint, Esprima) are correctly installed in the game's `MelonLoader/Libs` folder.
:globe_with_meridians: **Localization Update** — Improved support for community-driven translations.
:bug: **Fixes** — Resolved several UI glitches in the mod browser.

---

:package: **Installation**

1.  **Download**: Get the latest `GregModmanager-Installer.exe` from the [Releases](https://github.com/mleem97/gregModmanager/releases) page.
2.  **Run**: Execute the installer and follow the instructions.
3.  **Setup**: On first launch, point the manager to your *Data Center* installation directory.
4.  **Sync**: Click "Sync Core" to ensure your game is ready for modding.
