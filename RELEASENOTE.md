:package: **gregModmanager v1.1.0 is LIVE!**

Hey everyone,

The desktop companion for the Greg Framework has been updated! **gregModmanager v1.1.0** is now available, featuring major infrastructure improvements and better integration with the **gregStore** ecosystem.

---

:wrench: **What's new in v1.1.0?**

:cloud: **GitHub Mod Channel** — Directly install and update mods from GitHub repositories.
:lock: **BetterAuth Integration** — Unified login via `gregframework.eu`. Authenticate once, access all framework services.
:shield: **Git Verification** — Official members can now verify their status against the framework gitserver.
:steam: **SteamPlugin Support** — Integrated support for the DataCenter-SteamPlugin by ASavageSwan.
:sparkles: **Modern Branding** — Fully rebranded to **gregModmanager** and **gregStore** for a consistent ecosystem feel.

---

:package: **Installation**

1.  **Download**: Get the latest `gregModmanager-Installer.exe` from the [Releases](https://github.com/mleem97/gregModmanager/releases) page.
2.  **Run**: Execute the installer and follow the instructions.
3.  **Setup**: On first launch, point the manager to your *Data Center* installation directory.
4.  **Auth**: Log in using your gregFramework account to enable GitHub syncing.
